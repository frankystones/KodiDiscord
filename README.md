# KodiDiscord
Discord Rich Presence for Kodi

## How to use

* Make a TMDB account and copy your API key
* Download the repo
  * install `requirements.txt`
  * rename `.env.example` to `.env` 
  * fill in your TMDB API key
  * run `main.py`

## Supported Media Types

* TV Shows
* Movies
* Live TV (PVR IPTV Simple Client)
